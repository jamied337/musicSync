<h1 style="text-align: center;"><span style="text-decoration: underline;">musicSync</span></h1>
<p style="text-align: center;">simple script to sync music libraries</p>
<p style="text-align: left;">This script works by copying a library of .flac files to another location and converting them to .mp3 files while retaining metadata such as title, album, artist, track number, and album art.</p>
<p style="text-align: left;">&nbsp;</p>
<h2 style="text-align: left;"><strong>Dependancies:</strong></h2>
<ul>
<li>pathlib</li>
<li>mutagen</li>
<li>pydub</li>
</ul>
<p>&nbsp; &nbsp; Installation on most systems:</p>
<ul>
<li>
<p>pip install pathlib mutagen pydub</p>
</li>
</ul>
<p>&nbsp; &nbsp; Installation on arch-based systems:</p>
<ul>
<li>sudo pacman -S python-pathlib python-mutagen python-pydub</li>
</ul>
<p>&nbsp;</p>
<h2>Usage:</h2>
<ul>
<li>
<p>python3 musicSync.py</p>
</li>
</ul>
